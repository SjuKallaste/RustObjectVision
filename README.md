"# RustObjectVision"
A desktop (Windows / Linux) program made to segment the input image based on color or texture. Then calculate any segment's area.

In order to run this program on Windows just download the
```image-segementer.exe```
file ONLY from releases.

If you are running on Linux you must use the AppImage file

If you happen to be on MacOS or any other operating system you must have Rust installed and use
```cargo run --release```
in the project directory.
